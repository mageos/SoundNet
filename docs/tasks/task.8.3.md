# Task 8.3: Create build scripts and packaging instructions

**Milestone:** 8 - Finalization & Documentation
**Depends On:** 8.2
**Status:** Complete

## Description

Write a `build.sh` script for creating release builds (`cargo build --release`). Add instructions for cross-compiling for ARM SBCs (e.g., Raspberry Pi). If possible, create a basic `.deb` package or a systemd service file.

## Acceptance Criteria

- There is a clear, documented process for building the application for production and deploying it on a target device.
